# The Hairitage Barbershop

A modern, responsive static landing page for The Hairitage Barbershop in Farmington, Connecticut. Built with React, Vite, and Tailwind CSS for fast GitHub Pages deployment.

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start development server:

```bash
npm run dev
```

3. Open http://localhost:5173 in your browser.

## Build

```bash
npm run build
```

## Deploy to GitHub Pages

```bash
npm run deploy
```

The site will build into the `dist` folder and can be hosted as a static site.
